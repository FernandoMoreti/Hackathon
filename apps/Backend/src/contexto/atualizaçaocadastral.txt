---------------Detalhamento do Fluxo de Atualização Cadastral
A atualização de dados cadastrais é realizada exclusivamente via WhatsApp, 
com a intervenção de um atendente humano. O beneficiário inicia o processo fornecendo as informações necessárias, 
como o CPF e os dados a serem atualizados (ex: endereço, telefone). 
Em seguida, o beneficiário precisa enviar a documentação comprobatória (ex: comprovante de residência). Após o envio, 
o processo aguarda o atendimento humano, que é quem irá verificar as informações e conferir a documentação. 
Se as informações estiverem corretas, o atendente realiza a atualização no sistema (CRM). 
Caso contrário, ele informa o beneficiário sobre o erro ou a pendência, também via WhatsApp.

O fluxo de atualização cadastral é iniciado quando o beneficiário fornece os dados a serem alterados e anexa a documentação 
de suporte. Este processo é mediado pelo atendimento humano. O atendente tem o papel de conferir os documentos e verificar 
se as informações estão corretas. Se tudo estiver certo, ele realiza a atualização dos dados no sistema (CRM). 
A funcionalidade-chave é a verificação humana da documentação comprobatória para garantir a segurança e a precisão da atualização. 
Se houver qualquer erro na documentação, o atendente informa o beneficiário para que a correção seja feita, garantindo a conclusão 
do processo.

A atualização cadastral é feita via WhatsApp.

O beneficiário fornece os dados e envia a documentação comprobatória. (pode ser para modificar endereço, nome, status, nível do convênio)

O processo exige atendimento e conferência humana dos documentos.

O atendente usa o CRM para realizar a atualização.
